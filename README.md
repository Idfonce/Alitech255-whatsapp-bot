# 🤖 Alitech255 WhatsApp Bot

🇹🇿 **Made in Tanzania** 🇹🇿

A powerful WhatsApp bot with automatic features!

## ✨ Features

- ✅ **Always Online** - 24/7 connectivity
- ✅ **Auto Status React** - Automatically reacts to statuses
- ✅ **Auto View Status** - Auto-views all status updates
- ✅ **Auto Typing** - Shows typing indicator
- ✅ **Custom Commands** - !ping, !info, !menu, !owner

## 📱 Commands

| Command | Description |
|---------|-------------|
| `!ping` | Check if bot is alive |
| `!info` | Show bot information |
| `!menu` | Show all commands |
| `!owner` | Contact bot owner |
| `!status` | Bot status (owner only) |
